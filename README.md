# CodeBuddy

CodeBuddy is a student-focused productivity assistant built to make academic life more organized and less stressful. Managing assignments, deadlines, notes, and study materials across multiple apps can get overwhelming, so this project brings everything together into one smart workspace.

The idea behind CodeBuddy was to create something that students can actually use in their daily lives — not just for productivity, but also for studying smarter and staying consistent.

## Overview

CodeBuddy helps students manage their academic workflow more efficiently by combining task management with AI-powered study assistance. From tracking assignments to generating quick study summaries, the platform is designed to save time and improve focus.

This project was built with the intention of solving real problems students face every semester, especially when handling multiple subjects, deadlines, and study resources at once.

## Features

- Assignment and deadline tracking
- AI-generated study summaries
- Smart task organization
- Centralized academic workspace
- Simple and user-friendly interface
- Productivity-focused design
- Easy navigation and accessibility
